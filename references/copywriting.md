# Copywriting

## Title Principles
- Make the audience visible.
- Name the problem or benefit directly.
- Keep the title easy to understand in one glance.
- Prefer concrete phrasing over abstract value.
- Avoid exaggerated promises.

## Title Templates
```text
{目标人群}第一次做{主题}，先看这一篇
我把{主题}最容易踩的坑整理好了
别急着买{产品}，先确认这几点
{场景}真的很需要这份清单
用完{方法/工具}后，我才发现这件事
```

## Opening Hooks
```text
一开始我也以为{错误认知}。
后来才发现，真正影响结果的是{关键点}。
```

```text
如果你也正在{具体处境}，
这篇可以先帮你把顺序理清楚。
```

```text
这不是最全攻略，
但适合刚开始、不想踩坑的人。
```

## Body Patterns
### Practical Guide
```text
先说结论：
{结论}

适合：
{适合人群}

不太适合：
{不适合人群}

重点看这几项：
1. {重点1}
2. {重点2}
3. {重点3}
```

### Emotional Story
```text
最近一直卡在{具体状态}。

不是完全没做事，
只是每一步都很慢。

后来我试着先做{小动作}，
事情没有立刻变好，
但我终于能开始往下走了。
```

### Product Or Service Recommendation
```text
我更在意的是{真实需求}，
所以这次主要看了{判断标准}。

优点：
{优点}

需要注意：
{限制}

适合：
{适合人群}
```

## Comment Prompts
```text
你现在最卡的是哪一步？
你会先选哪一种？
你还想看哪个主题的整理？
有没有同样踩过这个坑的人？
评论区留一个关键词，我下篇继续整理。
```

## Hashtag Pattern
Use 8 to 12 tags:

```text
#{核心主题} #{目标人群} #{使用场景} #{内容类型} #{痛点关键词} #{搜索词} #{系列名}
```

## Avoid
- 你一定可以
- 全网最全
- 必看不看后悔
- 绝对有效
- 闭眼入
- 无脑冲
- 夸大功效或保证结果
