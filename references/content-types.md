## Selection Matrix
| Goal | Best workflow | Default structure |
| --- | --- | --- |
| 涨粉 | 情绪共鸣 / 故事轮播 | 1 cover + 6 to 12 pages |
| 收藏 | 攻略 / 清单 / 教程 | cover + steps + checklist |
| 转化 | 产品 / 服务 / 店铺种草 | pain point + proof + use case + CTA |
| 互动 | 评论征集 / 投稿 / 测试题 | hook + question + examples + comment prompt |
| 复盘 | 数据分析 / 爆款拆解 | observation + pattern + action list |
| 系列化 | 栏目 / 连载 / 模板化发布包 | recurring title + fixed page rhythm |

## 1. Image-text Carousel
Use for tutorials, guides, checklists, topic explainers, and structured advice.

Structure:

```text
00 cover: clear title and benefit
01 why this matters
02-04 key points or steps
05-07 examples or comparison
08-10 action checklist
final page: save/comment/follow prompt
```

## 2. Emotional Story Carousel
Use for共鸣, growth stories, personal reflection, graduation, study, work, relationships, or anxiety topics.

Structure:

```text
hook
specific scene
pressure becomes visible
contrast or self-doubt
small realization
gentle closing
```

## 3. Two-panel Comic Or Light Illustration
Use when the user wants visual storytelling but no private IP is provided. Use generic original subjects, simple humans, objects, or scene metaphors.

Structure:

```text
panel 1: reality, pain point, or awkward moment
panel 2: reaction, contrast, tiny solution, or emotional release
```

## 4. Single Poster
Use for quotes, announcements, event posters, wallpaper-like posts, or one-point reminders.

Structure:

```text
one key sentence
one visual subject
one supporting detail
minimal tags and strong save value
```

## 5. Product, Venue, Or Service Post
Use for种草, review, shop visit, tool recommendation, course or service positioning.

Structure:

```text
problem
why this product/place/service fits
real use scenario
details users care about
pros and limits
who should or should not use it
CTA
```

## 6. Tutorial Or Checklist
Use for practical content with high save value.

Structure:

```text
result preview
materials or prerequisites
steps
mistakes to avoid
checklist
next action
```

## 7. Personal Story Or Case Study
Use for authentic narrative, growth, career, study, life, or creator diary posts.

Structure:

```text
before
turning point
specific action
real obstacle
what changed
what I would do again
```

## 8. Short Video Script
Use for口播, vlog, tutorial, product demo, or visual narrative.

Structure:

```text
0-3s hook
3-8s context
8-25s core scenes or points
25-35s summary
final CTA or question
```

## 9. Operation And Interaction Post
Use for comment collection, user submissions, Q&A, giveaway-like engagement, and community follow-up.

Structure:

```text
clear participation topic
example comments
privacy reminder
what users will get
response rhythm
follow-up content plan
```
