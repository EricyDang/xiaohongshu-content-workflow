# 小红书内容创作工作流

## Core Use
Create complete Xiaohongshu-ready content packages. Keep outputs practical, platform-native, concrete, easy to publish, and adaptable to different accounts, audiences, products, and themes.

## Private IP Rule
- Do not include personal characters, private IP names, unpublished artwork, or distinctive character settings unless the user explicitly provides them and asks to use them.
- When a visual subject is needed, use generic placeholders such as `{主角}`, `{产品}`, `{地点}`, `{账号人设}`, or `{视觉主体}`.
- If no brand or character is provided, build a neutral original visual direction from audience, tone, scene, palette, and content goal.

## Workflow
1. Identify content goal:涨粉, 收藏, 转化, 种草, 互动, 复盘, or series building.
2. Define audience, theme, tone, account constraints, and visual assets.
3. Choose a workflow: carousel, emotional story, tutorial, list, two-panel comic, poster, product/venue post, video script, or operation post.
4. Extract concrete user pain points, search words, scenes, objections, and emotional hooks.
5. Build the structure: cover, page order, narrative arc, visual direction, and publish copy.
6. Write image prompts or design notes page by page.
7. Package the deliverable as a reusable Xiaohongshu publishing pack.
8. Run QA for readability, platform tone, image consistency, privacy, rights, and conversion path.

## Output Defaults
- Full publishing pack: 1 cover, 6 to 12 content pages, title options, body copy, comment prompt, hashtags, visual spec, image prompts, and QA checklist.
- Carousel images: vertical `1024x1536`, consistent layout, short readable Chinese text.
- Single poster: vertical `1024x1536` or square `1024x1024`, one clear message.
- Video script: hook, scene list, narration, captions, cover copy, and interaction prompt.
- Post copy: 1 main title, 3 to 5 backup titles, 1 body version, 1 comment prompt, 8 to 12 hashtags.

## Text Style
- Use specific, spoken, human Chinese.
- Prefer short sentences, concrete scenes, and visible details.
- Avoid empty slogans, exaggerated clickbait, and generic AI encouragement.
- Keep in-image text short. If text rendering matters, generate no-text images and add Chinese text afterward.
